# Portfolio (односторінковий сайт)

## Опис проєкту

Мета проєкту – створення прототипу портфоліо, яке надасть студенту можливість
легко і ефективно представити свої досягнення, проєкти та навички своєму
майбутньому роботодавцю.

Проєкт виконаний у форматі **односторінкового сайту** з адаптивною версткою та
інтерактивними елементами. Він включає секції про себе, проєкти, відгуки та
форму для співпраці.

---

## Базові задачі

- Адаптивна верстка для планшетів і десктопів.
- Дотримання семантики HTML5.
- Підключення кастомних шрифтів.
- Оптимізація растрової та векторної графіки.
- Підтримка retina-екранів.
- Підключення svg-іконок через sprite.
- Додавання фавікону.
- Hover-ефекти для інтерактивних елементів.
- Використання анімацій для покращення UX (необов’язково).
- Підключення API: [Документація](https://portfolio-js.b.goit.study/api-docs).

## Опис секцій

### Header

- Логотип та назва проєкту.
- Навігація по сайту з якірними посиланнями та плавним скролом.
- Кнопка “Order the project” веде до секції **Work together**.
- Для мобільних пристроїв навігація реалізована через бургер-меню з JS:
  - плавне відкриття на весь екран,
  - блокування скроллу,
  - закриття при натисканні на елемент або кнопку закриття.
- Прозорий фон, позиціонування через `absolute` поверх секції Hero.

### Hero

- Заголовок: "Hello. I’m Fullstack developer Lloyd Jefferson."
- Перелік соціальних мереж як список `<ul>` з відкриттям у новій вкладці.
- Контактний email через посилання `mailto:`.
- Фонове зображення у форматі PNG.

### About me

- Прихований заголовок для SEO.
- Опис себе та зображення як контент.
- Інформація про досвід та освіту у вигляді списку `<ul>` з аккордеоном.
- Перелік навичок реалізовано як горизонтальний слайдер (Swiper).

### Benefits

- Заголовок: "Benefits of working with me".
- Перелік переваг у списку `<ul>`.
- Кнопка “Order the project” веде до секції **Work together**.

### Projects

- Заголовок: "Projects".
- Перелік проєктів та опис у горизонтальному слайдері (Swiper).
- Посилання "See project" відкриває проєкт на GitHub у новій вкладці.

### FAQ

- Заголовок: "FAQ".
- Перелік популярних запитань у вигляді аккордеону `<ul>`.
- Всі елементи за замовчуванням закриті.

### Covers

- Прихований заголовок для SEO.
- Перелік обкладинок через `<ul>` або анімовані рядки при появі секції у
  viewport.
- Фонове зображення PNG або градієнт.

### Reviews

- Заголовок: "Reviews".
- Перелік відгуків у горизонтальному слайдері (Swiper).
- Дані завантажуються з backend.
- У разі помилки показується текст-заглушка "Not found".

### Work together

- Заголовок: "Let’s work together".
- Форма з обов’язковими полями `<input>` та `<textarea>` і кнопкою submit.
- Валідація email через `pattern`.
- Відправка POST-запиту на сервер при натисканні кнопки.
- Після успішної відправки відкривається модальне вікно і форма очищається.
- При помилці користувач отримує повідомлення, форма не очищається.
- Контактний номер телефону та email реалізовані через посилання.
- Фонове зображення PNG.

### Modal window

- Заголовок: "Thank you for your interest in cooperation!".
- Описова частина та кнопка закриття.
- Модальне вікно закривається по:
  - кнопці закриття,
  - кліку на backdrop,
  - клавіші Escape.
- При відкритому меню скролл сторінки блокується.
